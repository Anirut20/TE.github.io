สื่อการสอน: ไดโอด (Diode)

1. ความหมาย 2. หลักการทำงาน 3. คุณลักษณะ 4. โจทย์ตัวอย่าง 5. เครื่องมือตรวจสอบ

1. ความหมายของไดโอด

ไดโอด (Diode) คือ อุปกรณ์สารกึ่งตัวนำชนิดหนึ่งที่ทำหน้าที่สำคัญในการบังคับทิศทางการไหลของกระแสไฟฟ้าให้ไหลไปในทิศทางเดียว และป้องกันไม่ให้กระแสไฟฟ้าไหลย้อนกลับ เปรียบเสมือน "วาล์วน้ำกันกลับ" ในทางไฟฟ้า

ไดโอดมีขั้ว 2 ขั้ว คือ แอโนด (Anode, A) ซึ่งเป็นขั้วบวก และ แคโทด (Cathode, K) ซึ่งเป็นขั้วลบ โดยสัญลักษณ์ของไดโอดจะใช้ลูกศรเพื่อชี้ทิศทางที่กระแสไฟฟ้าสามารถไหลผ่านได้ (จาก A ไป K)

2. หลักการทำงาน

การทำงานของไดโอดจะขึ้นอยู่กับวิธีการจ่ายแรงดันไฟฟ้าให้กับขั้วทั้งสอง ซึ่งเรียกว่า "การไบอัส" (Biasing) แบ่งเป็น 2 สภาวะหลัก:

  • การไบอัสตรง (Forward Bias): คือการจ่ายแรงดันไฟฟ้าบวกเข้าที่ขั้วแอโนด (A) และจ่ายแรงดันลบเข้าที่ขั้วแคโทด (K) ในสภาวะนี้ ไดโอดจะอยู่ในสภาพ "เปิด" และยอมให้กระแสไฟฟ้าไหลผ่านได้ โดยจะมีแรงดันตกคร่อมตัวมันเล็กน้อย (ประมาณ 0.7V สำหรับไดโอดชนิดซิลิคอน)
  • การไบอัสกลับ (Reverse Bias): คือการจ่ายแรงดันไฟฟ้าลบเข้าที่ขั้วแอโนด (A) และจ่ายแรงดันบวกเข้าที่ขั้วแคโทด (K) ในสภาวะนี้ ไดโอดจะอยู่ในสภาพ "ปิด" และไม่ยอมให้กระแสไฟฟ้าไหลผ่าน (ในทางอุดมคติ)

3. คุณลักษณะและแบบจำลอง

ในการวิเคราะห์วงจร เรามักจะใช้แบบจำลองของไดโอดเพื่อทำให้การคำนวณง่ายขึ้น:

1. แบบจำลองอุดมคติ (Ideal Diode Model):

ไบอัสตรง: ทำหน้าที่เหมือนสวิตช์ปิด (Short Circuit), Vd = 0V

ไบอัสกลับ: ทำหน้าที่เหมือนสวิตช์เปิด (Open Circuit), Id = 0A

2. แบบจำลองแรงดันคงที่ (Constant Voltage Drop Model): เป็นแบบจำลองที่นิยมใช้และใกล้เคียงความจริงมากขึ้น

ไบอัสตรง: ทำหน้าที่เหมือนแหล่งจ่ายไฟขนาดเล็ก, Vd ≈ 0.7V (สำหรับไดโอดซิลิคอน)

ไบอัสกลับ: ทำหน้าที่เหมือนสวิตช์เปิด, Id = 0A

4. โจทย์ตัวอย่าง

โจทย์ข้อที่ 1

คำถาม: จากวงจรด้านล่าง หากแหล่งจ่ายไฟ (Vs) คือ 10V และตัวต้านทาน (R) คือ 1k โอห์ม จงหากระแสไฟฟ้าที่ไหลผ่านวงจร (ใช้แบบจำลองแรงดันคงที่ Vd = 0.7V)

เฉลย

วิธีทำ: 1. ไดโอดได้รับการไบอัสตรง จึงนำกระแส และมีแรงดันตกคร่อม 0.7V

2. หาแรงดันที่ตกคร่อมตัวต้านทาน: VR = Vs - Vd = 10V - 0.7V = 9.3V

3. หากระแสจากกฎของโอห์ม: I = VR / R = 9.3V / 1000Ω = 0.0093A

4. แปลงหน่วยเป็น mA: 0.0093A = 9.3mA

คำตอบที่ถูกต้อง: 9.3mA

โจทย์ข้อที่ 2

คำถาม: จากวงจรเดิม แต่กลับทิศทางของไดโอดดังภาพ จะมีกระแสไฟฟ้าไหลผ่านวงจรเท่าใด

เฉลย

วิธีทำ: ไดโอดได้รับการไบอัสกลับ จึงทำหน้าที่เหมือนสวิตช์เปิด วงจรจึงขาดและไม่มีกระแสไฟฟ้าไหล

คำตอบที่ถูกต้อง: 0A
